# MosaiX

MosaiX is a location-based social platform where users can discover and share unique, interesting places that may not be widely known. The goal is to help people find hidden spots worth exploring and places they believe deserve more attention.

## Concept

Users can create an account, log in, and browse posts near their current location. If they find a spot they think is special or unique, they can upload it to the app so other users can discover it too.

Other users can then:
- View nearby spots.
- Like posts.
- Save places for later.
- Get directions to a location.
- Leave comments such as “I’ve been there, and it’s a fantastic spot.”

## Features

- User authentication.
- Location-based post feed.
- Upload new spots with photos.
- Like and save posts.
- Comments on locations.
- Directions to a spot.
- Profile screen with editable description and avatar selection.

## Tech Stack

- React Native
- Expo
- Expo Router
- TypeScript
- Jest


## Getting Started

### Install dependencies

```bash
npm install
```

### Start the app

```bash
npx expo start
```

### Run tests

```bash
npm test
```
